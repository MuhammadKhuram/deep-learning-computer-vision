# Deep Learning for Computer Vision

This repository tracks my hands-on implementation of foundational and state-of-the-art Computer Vision models. The architectures are built locally in VS Code and trained using free cloud GPUs on Google Colab and Kaggle.

## 📊 Model Summary & Progress

| Model | Task Type | Framework | Dataset Used | Key Metric / Result |
| :--- | :--- | :--- | :--- | :--- |
| **AlexNet** | Classification | TensorFlow | CIFAR-10 | ~80% Test Accuracy |
| **VGGNet** | Classification | PyTorch | Custom Dataset | Training Complete |
| **ResNet** | Classification | TensorFlow | ImageNet-Mini | Exploring Residuals |
| **SSD** | Object Detection | PyTorch | Pascal VOC | In Progress |
| **YOLO** | Object Detection | PyTorch | Custom Objects | Upcoming |
